# 07-导航系统规范

## 文档信息
- **版本**: 1.0.0
- **最后更新**: 2025-12-21
- **依赖**: [01-设计令牌](./01-设计令牌.md), [04-按钮组件](./04-按钮组件.md)

## 概述

本文档定义SillyTavern导航系统的完整设计规范，包括顶部导航栏、侧边抽屉、底部导航、面包屑等所有导航相关组件。

## 2. 侧边抽屉 (Side Drawer)

### 2.1 基础结构

**HTML结构**:
```html
<div id="left-nav-panel" class="drawer">
  <div class="drawer-header">
    <h3>菜单</h3>
    <button class="drawer-close">
      <i class="fa-solid fa-times"></i>
    </button>
  </div>
  <div class="drawer-content">
    <!-- 抽屉内容 -->
  </div>
</div>
<div id="drawer-overlay"></div>
```

### 2.2 CSS定义

```css
.drawer {
  position: fixed;
  top: 0;
  left: 0;
  width: 300px;
  height: 100vh;
  height: 100dvh;
  background-color: var(--black70a);
  backdrop-filter: blur(var(--blurStrength));
  border-right: 1px solid var(--SmartThemeBorderColor);
  transform: translateX(-100%);
  transition: transform 0.3s cubic-bezier(0.4, 0, 0.2, 1);
  z-index: 200;
  display: flex;
  flex-direction: column;
  box-shadow: 4px 0 12px rgba(0, 0, 0, 0.3);
}

/* 打开状态 */
.drawer.open {
  transform: translateX(0);
}

/* 抽屉头部 */
.drawer-header {
  display: flex;
  align-items: center;
  justify-content: space-between;
  padding: var(--spacing-md);
  border-bottom: 1px solid var(--SmartThemeBorderColor);
  flex-shrink: 0;
}

.drawer-header h3 {
  font-size: var(--fontSize-xl);
  font-weight: 600;
  color: var(--SmartThemeBodyColor);
  margin: 0;
}

/* 关闭按钮 */
.drawer-close {
  width: 32px;
  height: 32px;
  background-color: transparent;
  border: none;
  border-radius: 5px;
  display: flex;
  align-items: center;
  justify-content: center;
  cursor: pointer;
  transition: all var(--animation-duration) ease;
}

.drawer-close:hover {
  background-color: var(--black50a);
}

.drawer-close i {
  font-size: var(--fontSize-lg);
  color: var(--SmartThemeBodyColor);
}

/* 抽屉内容 */
.drawer-content {
  flex: 1;
  overflow-y: auto;
  padding: var(--spacing-md);
}

/* 滚动条样式 */
.drawer-content::-webkit-scrollbar {
  width: 8px;
}

.drawer-content::-webkit-scrollbar-track {
  background: var(--black30a);
}

.drawer-content::-webkit-scrollbar-thumb {
  background: var(--black70a);
  border-radius: 4px;
}

/* 遮罩层 */

  position: fixed;
  top: 0;
  left: 0;
  width: 100vw;
  height: 100vh;
  background-color: rgba(0, 0, 0, 0.5);
  opacity: 0;
  visibility: hidden;
  transition: all 0.3s ease;
  z-index: 199;
}

  opacity: 1;
  visibility: visible;
}
```

## 8. Figma 使用指南

### 8.1 创建顶部导航组件

1. **创建导航栏容器**：
   - 宽度：Fill container
   - 高度：50px
   - 填充：`black70a` + 毛玻璃
   - 边框：底部1px

2. **创建Auto Layout**：
   - 水平排列
   - 间距：12px
   - 内边距：0 12px
   - 对齐：Space between

3. **创建变体**：
   - 属性：`Platform` = Desktop | Mobile

### 8.2 创建抽屉组件

1. **创建抽屉容器**：300x视口高度
2. **添加毛玻璃效果**
3. **创建变体**：Open | Closed
