# 02-颜色系统（Color System）

> 完整的颜色体系，包含主题色、语义色和透明度系统

---

## 颜色分类

### 1. 主题基础色

#### 深色主题（默认）

**背景色系**
- **主背景**: `#242425` (rgb(36, 36, 37))
  - 用途：页面默认背景
  - 应用：body 元素

- **模糊色调**: `#171717` (rgba(23, 23, 23, 1))
  - 用途：毛玻璃效果底色
  - 应用：抽屉、弹窗、导航栏

- **聊天背景**: `#171717` (rgba(23, 23, 23, 1))
  - 用途：聊天区域基础色调
  - 应用：消息容器背景

**文本色系**
- **主文本**: `#DCDCD2` (rgb(220, 220, 210))
  - 用途：主要内容文本
  - 对比度：高对比度，适合长文阅读

- **次要文本**: `#919191` (rgb(145, 145, 145))
  - 用途：辅助说明文本
  - 对比度：中等对比度

- **高亮文本**: `#E18A24` (rgb(225, 138, 36))
  - 用途：引用、强调、重点内容
  - 特征：暖色调，吸引注意

---

### 2. 灰度色阶

用于创建层次和深度感：

| 色阶 | 颜色值 | 用途 |
|------|--------|------|
| Grey 10 | `#191919` | 最深灰，用于深色背景 |
| Grey 30 | `#4B4B4B` | 深灰，用于禁用状态 |
| Grey 50 | `#7D7D7D` | 中灰，用于分隔线 |
| Grey 70 | `#AFAFAF` | 浅灰，用于次要图标 |
| Grey 75 | `#BEBEBE` | 更浅灰，用于提示文本 |

---

### 3. 语义化颜色

#### 状态颜色

**成功/激活**
- **主色**: `#58B600` (rgb(88, 182, 0))
  - 用途：连接成功、激活状态
  - 示例：API已连接图标

- **成功透明**: `rgba(0, 100, 0, 0.7)`
  - 用途：成功提示背景

**警告/错误**
- **警告主色**: `rgba(255, 0, 0, 0.9)`
  - 用途：警告提示、错误信息
  
- **纯红色**: `rgba(255, 0, 0, 1)`
  - 用途：删除按钮、危险操作

- **深红**: `rgba(100, 0, 0, 0.7)`
  - 用途：错误背景

- **深红悬停**: `rgba(150, 50, 50, 0.5)`
  - 用途：危险按钮悬停状态

**重要/优先**
- **优先色**: `#F44336` (rgb(244, 67, 54))
  - 用途：标记重要项、收藏

**信息提示**
- **信息蓝**: `rgba(100, 100, 255, 0.3)`
  - 用途：信息提示背景

**特殊色**
- **金色**: `#F8D300` (rgb(248, 211, 0))
  - 用途：星标、收藏图标

- **象牙色**: `#DCDCD2` (rgb(220, 220, 210))
  - 用途：浅色主题文本（保留）

---

### 4. 透明度系统

#### 黑色透明度阶梯

| 变量名 | 颜色值 | 不透明度 | 用途 |
|--------|--------|----------|------|
| black30a | `rgba(0, 0, 0, 0.3)` | 30% | 轻度遮罩 |
| black50a | `rgba(0, 0, 0, 0.5)` | 50% | 中度遮罩、边框 |
| black60a | `rgba(0, 0, 0, 0.6)` | 60% | 弹窗遮罩 |
| black70a | `rgba(0, 0, 0, 0.7)` | 70% | 重度遮罩 |
| black90a | `rgba(0, 0, 0, 0.9)` | 90% | 深色背景 |

#### 白色透明度阶梯

| 变量名 | 颜色值 | 不透明度 | 用途 |
|--------|--------|----------|------|
| white20a | `rgba(255, 255, 255, 0.2)` | 20% | 轻微高光 |
| white30a | `rgba(255, 255, 255, 0.3)` | 30% | 悬停高光 |
| white50a | `rgba(255, 255, 255, 0.5)` | 50% | 分隔线 |
| white60a | `rgba(255, 255, 255, 0.6)` | 60% | 次要文本 |
| white70a | `rgba(255, 255, 255, 0.7)` | 70% | 辅助文本 |

#### 灰色透明度

- `grey30a`: `rgba(50, 50, 50, 0.3)` - 灰色叠加
- `grey5020a`: `rgba(125, 125, 125, 0.2)` - 轻灰背景
- `grey5050a`: `rgba(125, 125, 125, 0.5)` - 中灰背景
- `grey7070a`: `rgba(175, 175, 175, 0.7)` - 滚动条颜色

---

### 5. 消息颜色系统

**用户消息**
- 背景色调: `rgba(0, 0, 0, 0.3)`
- 特征：深色，区分于AI消息

**AI消息**
- 背景色调: `rgba(60, 60, 60, 0.3)`
- 特征：略亮于用户消息

**系统消息**
- 颜色处理：降低饱和度和不透明度
- 特征：灰色调，非重点

## 颜色应用规则

### 文本颜色

```scss
// 主要内容
.text-primary {
  color: var(--SmartThemeBodyColor);
}

// 次要内容
.text-secondary {
  color: var(--SmartThemeEmColor);
}

// 强调内容
.text-emphasis {
  color: var(--SmartThemeQuoteColor);
}

// 禁用状态
.text-disabled {
  color: var(--grey50);
  opacity: 0.5;
}
```

### 背景颜色

```scss
// 主背景
.bg-primary {
  background-color: var(--SmartThemeBlurTintColor);
}

// 用户消息
.bg-user-message {
  background-color: var(--SmartThemeUserMesBlurTintColor);
}

// AI消息
.bg-bot-message {
  background-color: var(--SmartThemeBotMesBlurTintColor);
}
```

### 边框颜色

```scss
// 标准边框
.border-default {
  border-color: var(--SmartThemeBorderColor);
}

// 聚焦边框
.border-focus {
  border-color: var(--white100);
}

// 错误边框
.border-error {
  border-color: var(--warning);
}
```
