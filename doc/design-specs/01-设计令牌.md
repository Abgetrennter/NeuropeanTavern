# 01-设计令牌（Design Tokens）

> 设计令牌是设计系统的原子单位，定义了所有可复用的设计属性值

---

## 颜色令牌

### 主题颜色
```css
/* 主文本颜色 */
--SmartThemeBodyColor: rgb(220, 220, 210);

/* 强调文本颜色 */
--SmartThemeEmColor: rgb(145, 145, 145);

/* 引用/高亮颜色 */
--SmartThemeQuoteColor: rgb(225, 138, 36);

/* 模糊背景色调 */
--SmartThemeBlurTintColor: rgba(23, 23, 23, 1);
--SmartThemeChatTintColor: rgba(23, 23, 23, 1);

/* 用户消息背景色调 */

/* 边框颜色 */
```

## 间距令牌

### 基础间距
```css
/* 基于主字体大小的倍数 */
--spacing-xs: calc(var(--mainFontSize) * 0.25);   /* ~3.75px */
--spacing-sm: calc(var(--mainFontSize) * 0.5);    /* ~7.5px */
--spacing-md: calc(var(--mainFontSize) * 1);      /* ~15px */
--spacing-lg: calc(var(--mainFontSize) * 1.5);    /* ~22.5px */
--spacing-xl: calc(var(--mainFontSize) * 2);      /* ~30px */
--spacing-2xl: calc(var(--mainFontSize) * 3);     /* ~45px */
```

### 组件专用间距
```css
/* 底部表单 */
--bottomFormBlockPadding: calc(var(--mainFontSize) / 2.5);
--bottomFormIconSize: calc(var(--mainFontSize) * 1.9);
--bottomFormBlockSize: calc(var(--bottomFormIconSize) + var(--bottomFormBlockPadding));

/* 顶部导航 */
--topBarIconSize: calc(var(--mainFontSize) * 2);
--topBarBlockPadding: calc(var(--mainFontSize) / 3);
--topBarBlockSize: calc(var(--topBarIconSize) + var(--topBarBlockPadding));

/* 消息间距 */
--mes-right-spacing: 30px;
```

## 字体令牌

### 字体族
```css
--mainFontFamily: "Noto Sans", sans-serif;
--monoFontFamily: 'Noto Sans Mono', 'Courier New', Consolas, monospace;
```

### 字号系统
```css
/* 基础字号（可缩放） */
--fontScale: 1;
--mainFontSize: calc(var(--fontScale) * 15px);

/* 字号层级（基于主字号倍数） */
--fontSize-xs: calc(var(--mainFontSize) * 0.8);     /* ~12px */
--fontSize-sm: calc(var(--mainFontSize) * 0.9);     /* ~13.5px */
--fontSize-base: var(--mainFontSize);               /* ~15px */
--fontSize-lg: calc(var(--mainFontSize) * 1.2);     /* ~18px */
--fontSize-xl: calc(var(--mainFontSize) * 1.5);     /* ~22.5px */
--fontSize-2xl: calc(var(--mainFontSize) * 2);      /* ~30px */
```

## 阴影令牌

### 阴影控制

```css
text-shadow: 0px 0px calc(var(--shadowWidth) * 1px) var(--SmartThemeShadowColor);
```

## 模糊令牌

### 毛玻璃效果
```css
--blurStrength: 10;
--SmartThemeBlurStrength: calc(var(--blurStrength) * 1px);
```

### 模糊应用场景
```json
{
  "background-light": "blur(5px)",
  "background-medium": "blur(10px)",
  "background-heavy": "blur(20px)",
  "background-double": "blur(calc(var(--SmartThemeBlurStrength) * 2))"
}
```

## 动画令牌

### 动画时长

## 使用说明

### 在 Figma 中
1. 创建 Local Variables
2. 使用 Color 类型变量存储颜色令牌
3. 使用 Number 类型变量存储间距和字号

### 在 Sketch 中
1. 使用 Color Variables 功能
2. 创建 Shared Styles 引用这些变量
3. 保持命名与 CSS 变量一致
