## 概述

本文档定义SillyTavern弹窗系统的完整设计规范，包括模态对话框、确认框、提示框、弹出菜单、工具提示等所有弹窗相关组件。

## 1. 模态对话框 (Modal Dialog)

### 1.1 基础结构

**HTML结构**:
```html
<div class="modal-overlay">
  <div class="modal" role="dialog">
    <div class="modal-header">
      <h3 class="modal-title">标题</h3>
      <button class="modal-close">
        <i class="fa-solid fa-times"></i>
      </button>
    </div>
    <div class="modal-body">
      <!-- 内容区域 -->
    </div>
    <div class="modal-footer">
      <button class="button-secondary">取消</button>
      <button class="button-primary">确认</button>
    </div>
  </div>
</div>
```

### 1.2 CSS定义

```css
/* 遮罩层 */
.modal-overlay {
  position: fixed;
  top: 0;
  left: 0;
  width: 100vw;
  height: 100vh;
  background-color: rgba(0, 0, 0, 0.6);
  backdrop-filter: blur(4px);
  display: flex;
  align-items: center;
  justify-content: center;
  z-index: 1000;
  opacity: 0;
  animation: modal-overlay-fade-in 0.2s ease forwards;
}

@keyframes modal-overlay-fade-in {
  to {
    opacity: 1;
  }
}

/* 模态框容器 */
.modal {
  background-color: var(--black70a);
  backdrop-filter: blur(var(--blurStrength));
  border: 1px solid var(--SmartThemeBorderColor);
  border-radius: 12px;
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.5);
  max-width: 600px;
  width: 90%;
  max-height: 90vh;
  display: flex;
  flex-direction: column;
  opacity: 0;
  transform: scale(0.9) translateY(-20px);
  animation: modal-slide-in 0.3s cubic-bezier(0.4, 0, 0.2, 1) forwards;
  animation-delay: 0.1s;
}

@keyframes modal-slide-in {
  to {
    opacity: 1;
    transform: scale(1) translateY(0);
  }
}

/* 模态框头部 */
.modal-header {
  display: flex;
  align-items: center;
  justify-content: space-between;
  padding: var(--spacing-lg);
  border-bottom: 1px solid var(--SmartThemeBorderColor);
  flex-shrink: 0;
}

.modal-title {
  font-size: var(--fontSize-xl);
  font-weight: 600;
  color: var(--SmartThemeBodyColor);
  margin: 0;
}

/* 关闭按钮 */
.modal-close {
  width: 32px;
  height: 32px;
  background-color: transparent;
  border: none;
  border-radius: 5px;
  display: flex;
  align-items: center;
  justify-content: center;
  cursor: pointer;
  transition: all var(--animation-duration) ease;
}

.modal-close:hover {
  background-color: var(--black50a);
}

.modal-close i {
  font-size: var(--fontSize-lg);
  color: var(--SmartThemeBodyColor);
}

/* 模态框内容 */
.modal-body {
  flex: 1;
  padding: var(--spacing-lg);
  overflow-y: auto;
  color: var(--SmartThemeBodyColor);
  font-size: var(--fontSize-md);
  line-height: var(--line-height-relaxed);
}

.modal-body::-webkit-scrollbar {
  width: 8px;
}

.modal-body::-webkit-scrollbar-track {
  background: var(--black30a);
}

.modal-body::-webkit-scrollbar-thumb {
  background: var(--black70a);
  border-radius: 4px;
}

.modal-body::-webkit-scrollbar-thumb:hover {
  background: var(--SmartThemeQuoteColor);
}

/* 模态框底部 */
.modal-footer {
  display: flex;
  align-items: center;
  justify-content: flex-end;
  gap: var(--spacing-sm);
  padding: var(--spacing-lg);
  border-top: 1px solid var(--SmartThemeBorderColor);
  flex-shrink: 0;
}
```

### 1.3 模态框尺寸变体

```css
/* 小尺寸 */
.modal.small {
  max-width: 400px;
}

/* 中尺寸（默认） */
.modal.medium {
  max-width: 600px;
}

/* 大尺寸 */
.modal.large {
  max-width: 800px;
}

/* 全屏 */
.modal.fullscreen {
  max-width: 100vw;
  max-height: 100vh;
  width: 100%;
  height: 100%;
  border-radius: 0;
}
```

### 1.4 响应式设计

```css
@media screen and (max-width: 1000px) {
  .modal {
    max-width: 95%;
    max-height: 95vh;
  }
  
  .modal-header,
  .modal-body,
  .modal-footer {
    padding: var(--spacing-md);
  }
  
  .modal-title {
    font-size: var(--fontSize-lg);
  }
}
```

## 8. Figma 使用指南

### 8.1 创建模态框组件

1. **创建遮罩层**：全屏矩形，填充`rgba(0,0,0,0.6)`
2. **创建模态框容器**：
   - 宽度：600px
   - 填充：`black70a` + 毛玻璃
   - 圆角：12px
   - 阴影：0 8px 32px

3. **创建变体**：
   - 属性：`Size` = Small | Medium | Large
   - 属性：`Type` = Default | Confirm | Loading

4. **添加Auto Layout**：垂直排列，适应内容

### 8.2 创建Toast组件

1. **绘制Toast容器**
2. **添加变体**：Success | Error | Warning | Info
3. **配置自动消失动画**

### 8.3 导出JSON

```json
{
  "modal": {
    "overlay": {
      "background": "rgba(0, 0, 0, 0.6)",
      "backdropBlur": "4px"
    },
    "container": {
      "maxWidth": "600px",
      "background": "rgba(0, 0, 0, 0.7)",
      "backdropBlur": "20px",
      "borderRadius": "12px",
      "boxShadow": "0 8px 32px rgba(0, 0, 0, 0.5)"
    }
  },
  "toast": {
```
